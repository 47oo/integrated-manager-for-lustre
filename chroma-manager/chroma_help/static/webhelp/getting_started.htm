Getting Started

  

Chroma Manager can be used to:

•Manage and monitor a Chroma Storage software appliance that has been created using Chroma Manager.

•Monitor an existing Lustre file system.

Managing and Monitoring a Chroma Storage Appliance

To set up a system for managing and monitoring a Chroma Storage appliance, follow these steps:

1.For procedures for installing and configuring the Chroma Enterprise software on your Chroma Manager and Chroma Storage servers, refer to the documentation provided by your storage solution provider.

2.To finish configuring Chroma Manager, go to Configuring Chroma Manager.

3.To create a Lustre file system, go to Creating a Lustre File System.

4.To mount the the client compute servers, go to Mounting File System Clients.

To manage a Lustre file system that has been created in Chroma from the command line, you must use the Chroma Manager command line interface (CLI). Modifying the Lustre file system manually from a shell on a Chroma Storage server will interfere with the ability of Chroma Manager to manage and monitor the file system.

Monitoring Only

To monitor an existing file system, the chroma-agent  package must be installed on the storage servers in the system. The storage servers are then registered with the Chroma Manager to allow Chroma Manager to monitor performance and other metrics.

1.For a procedure to install chroma-agent on your storage servers, refer to the documentation provided by your storage solution provider.

2.To complete the setup, see Configuring an Existing Lustre File System.

Lustre commands can be used to manage metadata or object storage servers in an existing Lustre storage system that has been set up outside Chroma Manager and is being monitored, but not managed, in Chroma Manager.
