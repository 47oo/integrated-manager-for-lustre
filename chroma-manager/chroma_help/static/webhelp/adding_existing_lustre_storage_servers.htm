Adding Existing Lustre* Storage Servers


Contents

Note: All references herein to the manager refer to the Intel® Manager for Lustre* software.

A Lustre* file system that was created without using the manager can be monitored, but not managed, from the manager GUI. Before completing the procedure below, make sure that the chroma-agent executable has been installed on each of your Lustre file system storage servers. See the documentation provided by your storage solution provider for details.

Before an existing Lustre file system can be monitored at the manager GUI, the object storage servers must be added and detected by the manager. To add each object storage server, complete these steps:

1.Go to Configuration > Servers and click +Add server. The Add Server dialog is displayed.

2.In the Hostname field, enter the name of the server to be added.

3.Click Continue. The manager checks that the network connection to the server is available and that the chroma-agent service is correctly installed on the server.

4. Click Close.

5.To make the Lustre file system appear on the Dashboard and Configuration > File System pages in the manager GUI, complete these steps:

a.Go to Configuration > Servers and click Detect File Systems. A dialog listing hosts is displayed.

b.Select the hosts (servers) on which the targets for the file system to be detected are running and click Run. A Command detail dialog appears showing progress. Status shows “Successful” when the process is complete.

Note: To view the Command Detail after detection completes, click on Notifications on the left side of the screen and select Commands at the bottom of the notifications list. To the right of the Detecting file systems command, click Open.
 
Note: To be detected, the file system must be running.

The Lustre file system is now ready to be monitored at the manager GUI.
