Mounting File System Clients

  

A compute client must mount the Lustre file system to be able to access data in the file system.  Before attempting to mount the file system on your Lustre clients, make sure the Lustre client software has been installed on each client. For instructions, see the documentation provided by your storage solution provider.

To obtain the command to use to mount your file system, complete these steps:

1.Go to Configuration > File Systems.

2.In the table listing file systems, click the name of the file system to be mounted. A page showing information for that file system is displayed.

3.On the file system page, click Client mount information. The command used to mount the file system on a client is displayed.

4.On the client server, enter the command displayed in step 3.
