History window


Contents

The History page display events. The History page provides a finer level of event filtering than the listing of events displayed on the Status page.

On this page, you can filter to view:

●Up to the last 100 events

●Events occurring on all servers or one selected server.

●Any, or one of three levels of severity: Info, Warning, or Error.

●Any, or one of four possible event types: Autodetection, Alert, Syslog, or Storage resource detection. See also View alert and event status messages.
