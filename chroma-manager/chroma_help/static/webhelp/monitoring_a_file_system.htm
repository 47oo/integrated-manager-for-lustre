Monitoring a Lustre File System

  

A file system is monitored in the Chroma Manager using the dashboard and the alerts, events, and logs pages.

Dashboard

The dashboard displays a set of graphs that provide usage and performance data at several levels in the file systems being monitored.  At the top level, an aggregate view of all file systems monitored by Chroma Manager is displayed. Individual file systems and servers can be displayed using the selection fields at the top of the page.

Data used to produce the graphics is saved for long-term use. Data is averaged and compressed over time so that the most recent data is stored and viewed at maximum resolution while aging data is stored and viewed at progressively lower resolutions over time.

 

Alerts, events, and logs

Alerts – Shows active alerts and alert history.

Events – Displays a list of events and allows filtering of events by host, severity, and event type.

Logs –  Displays log information and allows filtering by host, date range, and messages from Lustre or all sources.
