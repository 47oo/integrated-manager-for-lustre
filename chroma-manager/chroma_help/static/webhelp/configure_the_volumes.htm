Configure primary and failover servers


Contents

To view the volumes that were discovered and make adjustments to volume configurations, complete these steps:

1.Go to Configuration > Volumes to display the Volume Configuration page. A list of available volumes is displayed (if a volume does not contain unused block devices, it will not appear on this list).

2.For a given volume, select the volume's Primary Server from the drop-down list. Then select the Failover Server from the drop-down list.

3.Repeat step 2 for each volume that has a primary and failover server.

4.Click Apply.

Remember these server configurations for when configuring power distribution units (PDUs) and outlet-server assignments.  Failover is supported by PDUs and outlet assignments or by assigning BMCs to servers.  See Add power distribution units or Assign BMCs to servers.
