Introducing Chroma Enterprise

  

The Chroma Enterprise scalable storage solution tightly integrates management, monitoring, and high availability features with Enterprise Lustre software to create a high performance global file system based on enterprise-class storage hardware. Chroma Enterprise comprises the Chroma Manager, one or more Chroma Storage software appliances, and enterprise-class storage from any storage solution provider that supports the Chroma Storage software, as shown below.

Chroma Enterprise scalable storage solution

The Chroma Storage software appliance runs on servers attached to enterprise storage hardware provided by a storage solution provider to form the Enterprise Lustre file system. Key components are the open source Enterprise Lustre software integrated with Linux CentOS and additional user libraries to enable a truly integrated software storage appliance. The Chroma Storage software appliance delivers fast IO to applications across an Infiniband or Ethernet network.

The Chroma Manager includes features to simplify the creation and management of the Enterprise Lustre file system using either a browser-based UI or a command line interface (CLI).

To manage Chroma Storage from the command line, you must run Chroma CLI commands on the Chroma Manager server. Do not run Lustre commands from the UNIX command line of a metadata or object storage server in a Chroma Storage appliance that is being managed using Chroma Manager.

Lustre commands can be used to manage metadata or object storage servers in an existing Lustre storage system that has been set up outside Chroma Manager and is being monitored, but not managed, in Chroma Manager.

The Chroma Manager also serves as a central vantage point from which to monitor the entire distributed Enterprise Lustre file system, with storage-monitoring features that enable monitoring of Lustre file system usage, performance metrics, and events or errors at the Lustre level.  Plugins provided by storage solution providers make hardware-level performance data, disk errors and faults, and other relevant hardware-related information available to the Chroma Manager.
