Adding Existing Lustre Storage Servers

  

A Lustre file system that was created outside Chroma manager can be monitored, but not managed, by Chroma Manager. Before completing the procedure below, make sure that the chroma-agent package has been installed on each of your Lustre file system storage servers. See the documentation provided by your storage solution provider for details.

Before an existing Lustre file system can be monitored by Chroma Manager, the object storage servers must be added to and detected by Chroma Manager.

To add each object storage server to Chroma Manager, complete these steps:

1.Go to Configuration > Servers and click +Add server. The Add Server dialog is displayed.

2.In the Hostname field, enter the name of the server to be added.

3.Click Continue. Chroma Manager checks that the network connection to the server is available and that the chroma-agent software is correctly installed on the server.

4. Click Close.

5.To make the Lustre file system appear on the Dashboard and Configuration > File System pages in Chroma Manager, complete these steps:

a.Go to Configuration > Servers and click Detect Filesystems. A dialog listing hosts is displayed.

b.Select the hosts (servers) on which the targets for the file system to be detected are running and click Run. A Command detail dialog appears showing progress. Status shows “Successful” when the process is complete.

Note: To view the Command Detail after detection completes, click on Notifications on the left side of the screen and select Commands at the bottom of the notifications list. To the right of the Detecting filesystems command, click Open.
 
Note: To be detected, the file system must be running.

The Lustre file system is now ready to be monitored in Chroma Manager.
