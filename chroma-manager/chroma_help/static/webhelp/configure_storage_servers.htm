Add storage servers


Contents

This procedure adds one or more storage servers.  Using SSH will let you add one server at a time.  Using HTTPS will let you add up to 128 servers. At least two servers are required for HA file systems.

Note: All authentication credentials are sent to the manager server via SSL and not saved to any disk.

To add a server to be used for the file system:

1.Go to Configuration > Servers  to display the Server  tab.

2.Click Add server.

3.The Add server dialogue window provides two methods to authenticate and add the server(s): SSH and HTTP. To use the HTTPS, perform steps 4a - 4 e.
To use SSH, skip to step 5.

4.Click the HTTPS tab.  This tab will generate a command line that includes a token.  You can run this command on each server that you wish to add to the file system.  All necessary software will be installed on the server and the server will be registered with Intel® Manager for Lustre.   Perform these steps:

a.Under Server profile, select Managed storage server.

b.Using the scroll bar, select the number of servers you want to add using the command.  You can select up to 128 servers.

c.Using the scroll bar, select the time period for which this command line will be functional.

d.Click Generate.

e.Enter the resulting command at each server you wish to add to a file system.  The manager checks that each server is up and connected to the network. It then does an audit of the storage resources on each server.  The manager then checks the networking, loads Lustre modules, and makes sure the Lustre networking layer is functioning. When all checks are completed, the server is fully qualified as a Lustre server. This process is complete. Proceed to Configure primary and failover servers.

5.Click the SSH tab (selected by default).

6.In the Hostname field, enter the name of the server to be added.

7.Under Server profile, select Managed storage server.

8.Under SSH Authentication, select on one of these three methods:

•Click Existing Key to use an existing SSH private key present on this management server.  There must be a corresponding SSH public key on server you are adding.

•Click Root Password and enter a root password for the server you are adding.

•Click Another Key and enter a private key that corresponds to a public key present on the server you are adding.  If the key is encrypted, enter the passphrase needed to decrypt the private key.

9.Click Continue. The manager checks that each server is up and connected to the network. It then does an audit of the storage resources on each server.  The manager then checks the networking, loads Lustre modules, and makes sure the Lustre networking layer is functioning. When all checks are completed, the server is fully qualified as a Lustre server.

10.To add another server, click Add Another.  Otherwise, click Close.

11.Proceed to Configure primary and failover servers.
