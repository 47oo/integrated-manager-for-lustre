Configure PDUs and Outlets to Support HA and Failover


Contents

Use this tool to add power distribution units (PDUs) and to assign specific PDU power outlets to individual servers.  Once configured, this feature lets you check the status of PDUs and individual outlets.  Based on server power requirements and your failover configuration, you may want to assign more than one outlet to a server.  For improved failover performance, consider assigning the failover outlet from a different PDU than the primary outlet.  When you associate PDU failover outlets with servers using this tool, STONITH is automatically configured.

Note: All references herein to the manager refer to the Intel® Manager for Lustre* software.

Add PDUs

1.Go to Configuration > PDUs.

2.With no power distribution units recognized, this tab will read: No power distribution units are configured.  Click Add PDU.

3.At the Add PDU dialogue, select the Device type from the drop-down list.

4.Enter a name for this PDU. (If not entered, this field will default to the IP address.)

5.Enter the IP address for the PDU. If not known, enter a DNS-resolvable hostname.  The address is stored as an IPv4 address. Note: If this IP address later changes in DNS, it will not automatically update here and you will need to update this address.

6.Enter the port number.  This port number must be unique to this PDU.

7.Enter your Management user name.

8.Enter your Management password. Then click Save.

Assign PDU outlets to servers

Note: Electrical connection between a given PDU outlet and a specific server is performed by a qualified technician before PDUs are configured and assigned by this software.

At the next screen, you can assign PDU outlets to servers.  The left column shows all servers used in all file systems that you're currently managing.   Each column to the right of the Server column shows outlet assignments for one PDU.  If you have four PDUs configured, then there are four PDU columns.

Each row represents an outlet-to-server assignment.  To assign PDU outlets to servers:

1.Pick a server row for which you want to assign outlets.

2.Mouse over to the PDU column and click within the drop-down box to expose the outlets available from this PDU.  Now select the desired outlet.  (You can also use the tab key to tab to the desired server/PDU.  Then begin to enter the outlet name. This field auto-fills.  Tab or press Enter to confirm this selection.)

3.Move to down to the next server and assign outlets in the same way.  Note that as an outlet is assigned to a server, it becomes unavailable for reassignment.

4.To remove an outlet from a server, click the <insert icon> next to the outlet name.   It now becomes available to reassign.

Remove an outlet

To remove the outlet from this server assignment, click the <insert icon> next to the outlet name.  This action makes this outlet available for assignment again (assuming the outlet is available).

Learn the IP address of a PDU outlet

Hover over the outlet icon <insert icon>.

Edit this PDU's parameters

Click <insert icon> to open a dialogue and edit the PDU's parameters.

Remove a PDU

Click  <insert icon> to remove the PDU.

Note: Removing the PDU will remove all monitoring of the PDU's outlets by the manager.
