OST Index Assignments Facilitate Striping


Contents

When selecting new OSTs for creating a new file system, or adding OSTs to an existing file system, Intel® Manager for Lustre* software will automatically assign OST indices in a distributed fashion across servers, to permit striping.  This capability requires at least two primary servers, with each hosting at least one OST.    Typical Lustre* file systems may have several primary servers and many OSTs.   Based on the number of primary servers and OSTs, the  manager will assign OST indexes in a distributed fashion to permit striping and improve system performance.
